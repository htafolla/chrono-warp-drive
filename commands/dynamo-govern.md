# /dynamo:govern

Submit a proposal to Dynamo for a governance verdict. Dynamo analyzes the proposal using 7D resonance analysis, Kuramoto coupling, and (optionally) real-time solar activity data.

**Usage:** `/dynamo:govern <proposal text>`

**Example:** `/dynamo:govern Should we deploy to production?`

**Tool invoked:** `governance` (or `govern_with_solar` for solar-modulated verdicts)
